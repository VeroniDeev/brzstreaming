## Run Locally

Clone the project

```bash
  git clone https://github.com/VeroniDeev/brzstreaming.git
```

Go to the project directory

```bash
  cd brzstreaming
```

Install dependencies

```bash
yarn install
```

or

```bash
npm install
```

Start the server

```bash
yarn run dev
```
